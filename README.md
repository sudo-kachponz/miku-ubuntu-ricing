## 🛠 Definitions: OS & Tools

* **Operating System (OS):** The base software that manages your computer's hardware and software resources. In this setup, you are using **Ubuntu 24.04.3 LTS (Noble Numbat)**, a Linux-based distribution known for its stability and ease of use.
* **Window Manager (Hyprland):** A dynamic tiling Wayland compositor. Unlike a traditional desktop environment, Hyprland manages how your windows are tiled and positioned on the screen, optimized for speed and visual effects.
* **AGS (Aylur's GTK Shell):** A framework for creating highly customizable, feature-rich desktop widgets (like your bars, dock, and calendar) using JavaScript and GTK.
* **Terminal (Foot):** A fast, lightweight, and minimalist Wayland-native terminal emulator.
* **Launcher (Wofi):** A flexible application launcher for Wayland, used here to open apps quickly via your custom Miku-themed button.
* **Shell (Zsh):** The interactive command-line interface, customized with themes and plugins to improve your productivity in the terminal.

## ✨ Key Features

* **Custom Calendar Popup:** A unique "Paint" UI-themed calendar featuring an interactive image slider for your Miku collection.
* **Tamagotchi-style Launcher:** A pixel-art inspired Start Menu button.
* **Aesthetic Integration:** A unified Cyan and Pink color palette across your bar, widgets, and terminal prompt.

## 🚀 Installation

To apply this configuration to your system:

1. **Clone the repository:**
```bash
git clone https://github.com/sudo-kachponz/miku-ubuntu-ricing.git
cd miku-ubuntu-ricing

```


2. **Deploy configuration:**
```bash
cp -r .config/* ~/.config/

```


3. **Restart the interface:**
```bash
killall ags && ags &

```
